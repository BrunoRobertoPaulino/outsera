Pré-requisitos  

Para executar o projeto é necessário ter instalado:
JAVA JDK 21  
APACHE MAVEN 3.9.6

Na pasta raiz do projeto, execute:   

mvn clean install  
mvn spring-boot:run

A aplicação será iniciada na porta padrão:    
http://localhost:8080  

Endpoint disponível  
Buscar intervalos de produtores premiados  
GET http://localhost:8080/movies
